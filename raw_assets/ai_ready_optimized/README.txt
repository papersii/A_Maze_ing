# AI Ready Optimized Assets Instructions
# AI已优化标准化素材说明

## Purpose (用途)
This folder contains the final, game-ready assets that have been standardized to the game's grid system (64x64 pixels for high detail).
此文件夹包含最终的、游戏就绪的素材，这些素材已标准化为游戏的网格系统（64x64像素以保留细节）。

## Pipeline Status (管线状态)
1. `ai_generated_raw` -> Original AI output.
2. `ai_processed_transparent` -> Background removed.
3. `ai_ready_optimized` -> **HERE**: Resized, centered, and padded to 16x16.

## Next Step (下一步)
These assets are ready to be packed into the final Texture Atlas (e.g., `assets/sprites/objects.png` or new atlas) by the texture packer tool.
